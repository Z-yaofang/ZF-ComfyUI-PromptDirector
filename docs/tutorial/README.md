# ZF 创意导演教学视频制作包

这套资料按“两张接法图、一个核心主题”整理：重点讲清提示词创意导演如何把用户提示、世界观、用途、视觉方法和参考图创意组织成可执行的生图任务。视频中不分发额外教学工作流，也不把模型安装当成主角；提示词任务生成后，再用同一份结果对比本地成图模型与闭源/API模型。

## 两张接法图怎么讲

1. **本地模型接法图**：只展示导演台、参考图分析、本地写作节点和成图分支。重点是“提示词任务已经生成”，模型文件只作为一个可替换的执行端。
2. **API 接法图**：展示 API①读取参考图、导演台组织任务、API②逐任务写成最终提示词。重点是两个 API 的职责和先后关系，不演示密钥配置细节。

两条路线共用同一套导演台输入和同一份演示提示。先让观众看到任务列表与最终提示词，再分别把它送入本地模型和闭源/API模型，比较语义理解、细节执行和成图稳定性。

## 需要准备的演示素材

- 一张主体、构图和材质关系清楚的参考图，不含隐私、二维码或未授权品牌素材；
- 一段简短用户提示，指定成年主体、地点、动作和画面目标；
- 一个现实世界观和一个科幻/幻想世界观，用于演示扩展与替换；
- 一组匹配的“用途 + 视觉方法”，以及一组故意冲突的组合；
- 同一份最终提示词在本地模型和闭源/API模型中的成图结果。

## 录屏参数

- 画布：2560×1440；
- 帧率：30 fps；
- OBS 使用 MKV，录制后再封装为 MP4；
- 浏览器只放大当前节点区域，不要展示整个巨大工作区；
- 声音 48 kHz，轻度降噪和限幅即可。
