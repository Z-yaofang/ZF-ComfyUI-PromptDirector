# 录屏镜头表

| 镜头 | 时长 | 画面操作 | 屏幕提示 |
|---|---:|---|---|
| 01 | 15 秒 | 本地与闭源/API成品并排出现 | 同一提示词，不同执行端 |
| 02 | 35 秒 | 展示本地接法图全景，再放大导演台区域 | 输入相同，写作端可替换 |
| 03 | 35 秒 | 展示 API 接法图，依次指向 API①、导演台、API② | API①读图，API②写作 |
| 04 | 60 秒 | 放大用户提示、世界观、空文本和用途创意 | 五类输入各司其职 |
| 05 | 60 秒 | 运行导演台，先展示任务列表再展示系统提示词 | 先查任务，再查模型 |
| 06 | 70 秒 | 接入参考图分析与临时创意适配器 | 提取机制、元素、构图和材质 |
| 07 | 60 秒 | 展示一条完整最终提示词 | 任务已变成可执行指令 |
| 08 | 90 秒 | 同一提示词送入本地成图模型 | 离线、成本、执行差异 |
| 09 | 90 秒 | 同一提示词送入闭源/API模型 | 语义理解与细节执行差异 |
| 10 | 60 秒 | 对照两张结果，回到提示词而非乱改接线 | 模型差异不等于插件失效 |
| 11 | 45 秒 | 展示世界观冲突、参考图 latent、故事分镜错误截图 | 三个最常见误区 |
| 12 | 30 秒 | 回到 GitHub 仓库和素材来源说明 | 下载插件，按自己的工作流接入 |

## 录屏方法

- 每个模块单独录制，模型加载和 API 等待只保留开始与完成画面；
- 节点文字看不清时用后期局部放大，不要在录制中反复拖动整张工作流；
- 关键接口用矩形高亮，避免遮住输入名和连线；
- 对比模型时锁定提示词、尺寸、数量和种子，尽量只改变成图执行端；
- 错误案例使用已有截图，避免为了复现错误破坏当前工作流。
