## 视频简介

本期用两张接法图介绍 ZF-ComfyUI-PromptDirector 的核心提示词功能，包括：

- 用户提示、世界观、用途与视觉方法的职责；
- 明确的空文本路线；
- 参考图核心创意、元素、构图和材质提取；
- 本地写作路线与 API①视觉分析、API②最终提示词写作的职责；
- 同一份最终提示词交给本地模型和闭源/API模型后的差异；
- 参考图语义迁移、空文本、世界观扩展和常见错误判断。

插件与教学资料：

- PromptDirector：<https://github.com/Z-yaofang/ZF-ComfyUI-PromptDirector>
- Helper：<https://github.com/Z-yaofang/ZF-ComfyUI-Helper>

素材来源与致谢：插件当前及后续版本中使用、整理和扩展的用途、创意方向、案例启发和相关提示词素材，来源于 **远古大呲花** 的素材整理与分享。B站主页：<https://space.bilibili.com/59550296>。插件代码、结构化整理、工作流适配与 ComfyUI 节点实现由 ZF-ComfyUI-PromptDirector 仓库完成；完整中英文说明见仓库首页。

注意：本期不提供新的教学工作流，演示使用现有工作流和两张接法图。不同语言模型和图像模型的语义执行能力不同，结果存在正常差异；视频中不展示 API 密钥、账户或私人路径。
